## 技术栈

| 技术 | 用途 |
|------|------|
| Flutter | 跨平台 UI 框架 |
| Riverpod | 状态管理 |
| Hive | 本地存储 |
| InAppWebView | 在线阅读 |
| http | HTTP 请求 |
| flutter_slidable | 滑动操作 |
| cached_network_image | 图片缓存 |
