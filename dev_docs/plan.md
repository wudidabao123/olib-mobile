## 1. 用户流程分析

### 1.1 当前用户旅程

```
启动 → Splash → [已登录?]
                    ├─ 是 → Home (热门/推荐)
                    └─ 否 → Login → (注册) → Home

Home → 搜索/筛选 → 书籍列表 → 书籍详情 → 下载/阅读
                                         ↓
                                    下载管理 → 打开文件
```

### 1.2 主要交互点

| 交互点 | 当前实现 | 用户体验 |
|--------|---------|----------|
| 登录 | 邮箱 + 密码 | ⭐⭐⭐ 基本 |
| 搜索 | 关键词 + 高级筛选 | ⭐⭐⭐⭐ 良好 |
| 书籍详情 | 信息展示 + 操作按钮 | ⭐⭐⭐ 基本 |
| 下载 | 点击下载 → 等待完成 | ⭐⭐⭐ 基本 |
| 收藏 | 添加/移除 | ⭐⭐ 需改进 |

## 2. 发现的交互问题

### 2.1 🔴 启动流程

**问题**: Splash 页面重试逻辑简单，网络超时处理不友好

**现状代码** (`splash_screen.dart`):
```dart
if (!_networkOk) {
  setState(() => _statusText = 'Network unavailable, retrying...');
  await Future.delayed(const Duration(seconds: 2));
  _networkOk = await _checkNetwork();  // 只重试一次
}
```

**建议**: 
- 增加重试次数上限 (3 次)
- 提供手动重试按钮
- 显示域名切换选项

---

### 2.2 🔴 登录页面更新检查重复

**问题**: 在 Login、Home、Settings 三个页面都会检查更新，用户可能收到多次更新提示

**建议**: 
- 在 Splash 页面或 App 初始化时统一检查
- 使用全局状态记录"本次会话是否已提示更新"

---

### 2.3 🟡 搜索体验

**问题**: 
1. 筛选面板默认展开，占用大量屏幕空间
2. 无搜索历史记录
3. 无热门搜索推荐

**建议**:
1. 筛选面板默认折叠
2. 添加搜索历史记录 (本地存储)
3. 在输入框空状态时显示热门搜索词

---

### 2.4 🟡 书籍详情页

**问题**:
1. 下载按钮状态变化不明显
2. 已下载书籍无法直接打开，需要去下载页面
3. 收藏状态同步不及时

**建议**:
1. 下载按钮增加更明显的状态指示 (进度环/已下载图标)
2. 已下载时显示"打开"按钮替代下载按钮
3. 使用 Riverpod 实时监听收藏状态

---

### 2.5 🟡 下载管理

**问题**:
1. 没有批量操作功能
2. 文件打开失败时提示不明确
3. 没有下载队列优先级设置

**建议**:
1. 添加长按多选 → 批量删除
2. 文件打开失败时提供"选择其他应用打开"选项
3. (可选) 支持拖拽排序下载队列

---

### 2.6 🟡 设置页面

**问题**:
1. 部分文本硬编码，未使用国际化系统
2. 使用 `_isZhLocale()` 判断而不是国际化 key

**现状代码**:
```dart
Text(_isZhLocale(context) ? '下载' : 'Downloads')
```

**建议**:
```dart
Text(AppLocalizations.of(context).get('downloads'))
```

---

### 2.7 🟢 收藏功能

**问题**:
1. 收藏成功无明显反馈
2. 收藏列表无排序选项

**建议**:
1. 添加收藏成功的 SnackBar 提示
2. 支持按添加时间/书名排序

---

### 2.8 🟢 阅读器

**问题**:
1. WebView 阅读器功能较基础
2. 无阅读进度保存
3. 无夜间模式切换

**建议** (长期):
1. 集成 EPUB 本地阅读器 (如 epub_view)
2. 保存阅读进度到本地
3. 阅读器内置亮度/主题切换

## 3. 优化建议

### 3.1 短期优化 (1-2 周)

| 优化项 | 文件 | 工作量 |
|--------|------|--------|
| 统一更新检查逻辑 | `update_service.dart` + 各页面 | 中 |
| 筛选面板默认折叠 | `search_screen.dart` | 小 |
| 已下载书籍直接打开 | `book_detail_screen.dart` | 中 |
| 收藏成功提示 | `book_detail_screen.dart` | 小 |
| 修复硬编码文本 | `settings_screen.dart` | 小 |

### 3.2 中期优化 (1 个月)

| 优化项 | 描述 | 工作量 |
|--------|------|--------|
| 搜索历史 | 本地存储最近搜索词 | 中 |
| 下载批量操作 | 长按多选 + 批量删除 | 中 |
| Splash 重试优化 | 手动重试 + 域名切换 | 中 |

### 3.3 长期优化 (3+ 个月)

| 优化项 | 描述 | 工作量 |
|--------|------|--------|
| 本地 EPUB 阅读器 | 集成 epub_view 或类似库 | 大 |
| 阅读进度同步 | 本地/云端进度保存 | 大 |
| 书架分类 | 自定义书架/文件夹 | 大 |

## 5. 技术债务

| 项目 | 描述 | 影响 |
|------|------|------|
| 混合存储后端 | SharedPreferences + Hive 混用 | 维护复杂 |
