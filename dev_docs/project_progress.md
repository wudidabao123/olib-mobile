## 文件结构

```
lib/
├── constants/          # 常量定义
├── l10n/              # 国际化文件
├── models/            # 数据模型
│   ├── book.dart
│   ├── download_task.dart
│   └── ...
├── providers/         # 状态管理
│   ├── auth_provider.dart
│   ├── books_provider.dart
│   ├── domain_provider.dart
│   ├── download_provider.dart
│   ├── settings_provider.dart
│   ├── theme_provider.dart
│   └── zlibrary_provider.dart
├── routes/            # 路由定义
├── screens/           # 页面
│   ├── auth/          # 认证
│   ├── book_detail/   # 书籍详情
│   ├── debug/         # 调试工具
│   ├── downloads/     # 下载管理
│   ├── favorites/     # 收藏
│   ├── home/          # 首页
│   ├── reader/        # 阅读器
│   ├── search/        # 搜索
│   ├── settings/      # 设置
│   ├── similar/       # 相似书籍
│   └── splash/        # 启动页
├── services/          # 服务层
├── theme/             # 主题配置
└── widgets/           # 公共组件
```
